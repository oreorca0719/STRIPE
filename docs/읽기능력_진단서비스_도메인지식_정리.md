## 2. 핵심 교육 메커니즘 1 — 읽기 능력의 2층 구조 진단

연구계획서에서 채택한 진단 프레임워크의 핵심은 **"유창성만으로 읽기 능력을 판정하면 오진이 발생한다"** 는 원칙이다. (박창균 외, 2019)

### 2-1. 1층: 읽기 유창성 (Reading Fluency)

읽기 유창성은 **속도, 정확성, 표현성**을 결합한 복합 능력이며, 음독(소리내어 읽기)과 묵독(소리 없이 읽기)으로 구분된다.

#### 음독 유창성 (Oral Reading Fluency)
| 측정 항목 | 설명 | 측정 방법 |
|-----------|------|-----------|
| 자동성 (Automaticity) | 글자를 빠르게 자동으로 해독하는 능력 | 10초당 정확하게 읽은 음절 수 = (정확 음절 수 ÷ 전체 소요시간) × 10 |
| 정확성 (Accuracy) | 오류 없이 읽는 능력 | 오류율 = (생략+대치+첨가+반복+수정 총 오류 수) ÷ 총 음절 수 |
| 표현성 (Prosody) | 억양·리듬·강세를 살려 읽는 능력 | 관찰자 체크리스트 또는 루브릭 평가 |

**측정 절차 (표준)**:
1. 대상자에게 텍스트 제시
2. 제목 읽기 시작 순간부터 타이머 시작
3. 마지막 음절 읽는 순간 타이머 종료
4. 오류 유형별로 기록
#### 묵독 유창성 (Silent Reading Fluency)
| 측정 항목 | 설명 | 측정 방법 |
|-----------|------|-----------|
| 자동성 | 눈으로 빠르게 글을 처리하는 능력 | 시작 신호 후 마지막 마침표에 '/' 표시하는 시간 측정 |
| 이해 | 소리 없이 읽은 내용 파악 능력 | 독해 문항 응답 (지문 없이) |

**중요 제약**: 묵독의 경우 정확도와 운율을 수량화하는 데 기술적 한계가 있음 → **속도 중심 측정** 채택 (학년 간 차이를 보여주는 데는 속도가 핵심 역할)

#### 학년별 음독 유창성 발달 기준 (참고값)
| 학년 | 이야기글 (분당 어절) | 설명글 (분당 어절) |
|------|---------------------|------------------|
| 초등 1학년 | 기준 설정 필요 | 약 69 어절/분 |
| 초등 3학년 | 1학년과 유의미한 차이 | 약 67 어절/분 |
| 초등 5학년 | 3학년과 유의미한 차이 없음 | 약 103 어절/분 |

> ⚠️ **IT 구현 시 주의**: 음독 유창성의 정확한 측정을 위해서는 **음성 입력(STT) 기술**이 필요하다. STT 없이는 자기보고 방식이나 교사/부모 대리 입력 방식으로 대체해야 하며, 이는 정확도를 떨어뜨린다.

### 2-2. 2층: 읽기 이해 (Reading Comprehension) — 3층위 구조

독해 이해는 사실적 → 추론적 → 비판적 이해의 3단계로 위계화된다.

#### 사실적 이해 (Literal/Factual Comprehension)
- **정의**: 글에 명시적으로 드러난 정보를 있는 그대로 정확하고 신속하게 이해하는 능력
- **하위 기능**:
  - 세부 정보 재인/회상
  - 주요 아이디어 파악
  - 인과 관계 확인 (명시적인 것만)
  - 비교/대조 정보 확인
  - 글의 구조 및 전개 방식 파악
- **평가 문항 예시**: "이 글에서 ○○이 발생한 직접적인 원인은 무엇입니까?"
- **특성**: 추론적/비판적 이해의 **기초** 역할, 낮은 인지 수준이지만 현대사회에서 필수
#### 추론적 이해 (Inferential Comprehension)
- **정의**: 글에 드러나지 않은 필자의 의도, 숨은 주제, 논리적 함의를 추론하는 능력
- **하위 기능**:
  - 글 내부 논리성 추론
  - 배경지식과 글 내용의 연관성 파악
  - 글에 드러나지 않은 원인/결과 추론
  - 필자의 의도/목적 추론
- **평가 문항 예시**: "작가가 이 이야기를 쓴 의도는 무엇이라고 생각합니까?"
- **핵심 차이**: 텍스트 **내부**에서만 정보를 가져오면 사실적 이해, 텍스트 **외부**(배경지식, 맥락)와 연결하면 추론적 이해
#### 비판적 이해 (Critical Comprehension)
- **정의**: 글의 내용, 조직, 표현에 대한 신뢰성·타당성·적절성을 판단하며 읽는 능력
- **하위 기능**:
  - 글의 논리적 타당성 평가
  - 필자의 편견·관점·의도 평가
  - 정보의 사실/의견 구분 판단
  - 글의 가치·효용성 평가
- **평가 문항 예시**: "이 글에서 필자의 주장이 타당하다고 생각합니까? 이유를 쓰시오."
- **오해 주의**: 비판적 이해 = 글을 부정적으로 보는 것이 아니라, **근거에 기반한 판단과 평가**
#### GRI (General Reading Inventory) 검사 프레임
연구계획서에서 참조한 이은주(2026)의 측정 도구로, **25문항** 구성
- 사실적·추론적·비판적 3층위를 체계적으로 측정
- 중학생 한국어 독자를 대상으로 개발
- **구현 시 핵심 이슈**: 실제 문항 원문과 라이선스 확인 필요

## 3. 핵심 교육 메커니즘 2 — PISA 읽기 평가 프레임워크

PISA(Programme for International Student Assessment)는 OECD가 만 15세 학생을 대상으로 3년 주기로 실시하는 국제 학업성취도 평가.

### PISA 2022 읽기 평가의 3대 차원

#### 차원 1: 과정 (Processes)
| 과정 | 설명 |
|------|------|
| 정보 찾기 | 텍스트에서 필요한 정보를 검색/탐색 |
| 이해하기 | 전반적 이해, 세부 이해, 추론적 이해 |
| 평가와 성찰하기 | 내용 평가, 형식 평가, 비판적 성찰 |
| 유창하게 읽기 | 빠르고 정확하게 읽는 능력 |
| 과제 관리 | 목표 설정, 계획 수립, 점검·조정 |

#### 차원 2: 텍스트 (Text)
| 분류 | 종류 |
|------|------|
| 출처 | 단일 출처, 다중 출처 |
| 체재 | 연속(산문), 비연속(표/그래프 등), 혼합 |
| 조직과 탐색구조 | 정적, 동적 |
| 유형 | 서사, 설명, 논증, 기술, 지시, 상호작용, 복합 |

**정답률 순서** (어려운 순): 복합 > 논증 > 설명 > 서사 > 기술 > 지시 > 상호작용

#### 차원 3: 상황 (Situations)
- 개인적, 공적, 교육적, 직업적 상황
### PISA 성취 수준 (읽기)
| 수준 | 특성 |
|------|------|
| 6수준 | 최상위, 복잡한 추론·비판적 평가 가능 |
| 3수준 (최다 분포) | 복잡한 텍스트에서 상세 정보 추론 |
| **2수준 미만** | **기초 학습 미달 — 이 서비스의 핵심 지원 대상** |

## 4. 핵심 교육 메커니즘 3 — 독자 유형 분류 체계

### 4-1. 이순영(2021b)의 3분법

| 유형 | 정의 | 특징 |
|------|------|------|
| **애독자** | 독서 빈도가 높고 독서 태도가 긍정적인 독자 | 독서의 내적 가치를 중시, 자발적 독서, 자기 주도적 책 선택 |
| **간헐적 독자** | 때때로 읽지만 일관된 독서 습관이 없는 독자 | 외부 자극(숙제, 추천)에 의존, 완독 경험 불규칙 |
| **비독자** | 책을 거의 읽지 않는 독자 | 독서 동기 결여, 한번 전환되면 개인 차원 회복 어려움 |

### 4-2. 비독자 전환 메커니즘

```
유치원 (독서 흥미 최고점)
        ↓ 하향 이동 추세
초등 저학년 (이 구간이 핵심 개입 가능 시기)
        ↓
초등 중학년 (비독자 전환 시작점)
        ↓ (한번 전환되면 회복 매우 어려움)
초등 고학년 / 중학교 (비독자 고착화)
```

### 4-3. 처방 매트릭스 구조

독자 유형 × 읽기 수준 = 처방 경로

```
              읽기 수준 (低) | 읽기 수준 (中) | 읽기 수준 (高)
애독자         A-1            A-2             A-3
간헐적 독자    B-1            B-2             B-3
비독자         C-1            C-2             C-3
```

**예시 처방 방향**:
- **C-1 (비독자 + 읽기 수준 낮음)**: 극도로 쉬운 텍스트, 장르(그림책/웹툰 형식), 완독 성공 경험 설계에 집중
- **A-3 (애독자 + 읽기 수준 높음)**: 읽기 수준 대비 약간 도전적인 텍스트, 다양한 장르 확장

## 5. 핵심 교육 메커니즘 4 — 텍스트 난도와 적합도서 원칙

### 5-1. '추천도서' vs '적합도서'

| 구분 | 추천도서 | 적합도서 |
|------|----------|----------|
| 기준 | 외부 전문가/기관이 좋다고 선정한 책 | 해당 독자의 특성에 맞는 책 |
| 문제 | 독자 개인 차이 무시 | - |
| 연구 근거 | 비독자 대상 도서 추천·홍보 → 오히려 독서 행동 하락 (이순영 외, 2024) | 독자-텍스트 특성 매칭이 핵심 (이순영, 2011) |

### 5-2. 텍스트 난도에 영향을 미치는 독자 요인 (이순영, 2011)

동일한 텍스트라도 독자에 따라 난도가 다르게 인식됨:

| 독자 요인 | 내용 |
|-----------|------|
| **흥미와 재미** | 관심 없는 텍스트 → 독서 동기 차단 + 독해 어려움 (가장 핵심 요인) |
| 배경지식 | 배경지식이 부족한 텍스트는 어렵게 인식 |
| 장르 선호 | 좋아하는 장르는 쉽게 느껴짐 |
| 독서 목적 | 자유 독서 vs 학습 독서 상황에 따라 난도 인식 차이 |

**학교급별 텍스트 난도 인식 차이**:
- **초등 저학년**: 텍스트 길이, 글씨 크기, 그림 유무로 판단
- **초등 고학년**: 이해와 공감 여부로 판단
- **중학생 이상**: 자신의 수준·목적에 따라 동일 텍스트도 다르게 인식
### 5-3. 이은주(2026)의 텍스트 선정 7원칙

텍스트 풀 구성 시 아래 7원칙을 충족해야 유효한 진단 결과 도출 가능:

1. **배경지식 통제**: 특정 학생에게 유리한 배경지식이 필요한 텍스트 배제
2. **문화 편향 배제**: 특정 문화권에 익숙한 내용 편향 제거
3. **장르 2종 이상 포함**: 이야기글(서사)과 설명글(정보) 최소 각 1종
4. **학년별 적정 어휘 수준**: 학년 수준에 맞는 어휘 사용
5. **적정 텍스트 길이**: 측정 목적에 맞는 분량 통제
6. **독립성**: 텍스트 간 내용 중복·연관성 배제 (공정한 개별 측정)
7. **중립성**: 특정 성별·지역·관심사에 편향되지 않도록
> ⚠️ **AI 활용 한계**: 구조화된 프롬프트 없이는 AI의 진단 품질이 제한적 (이순영·김주환, 2024). AI가 텍스트를 분석할 때는 감정이나 선입견 없이 객관적으로 평가할 수 있지만, 진단 품질은 **프롬프트 설계 품질**에 크게 의존

## 6. 서비스 기능 구조 — 3개 모듈

### 6-1. 진단 모듈 (RQ1)

```
[학습자 입력]
    │
    ├── 유창성 진단 (1층)
    │       ├── 음독 유창성: 자동성 / 정확성 / 표현성
    │       └── 묵독 유창성: 자동성 / 이해
    │
    └── 독해 이해 진단 (2층)
            ├── 사실적 이해
            ├── 추론적 이해
            └── 비판적 이해
                    └── GRI 기반 25문항 (라이선스 확인 필요)
```

**핵심 이슈 — 음독 측정 방식 결정 필요**:
| 방식 | 장점 | 단점 |
|------|------|------|
| STT(음성인식) 기반 자동 측정 | 정확하고 객관적 | 기술 구현 비용 높음, 소음 환경 취약 |
| 교사/부모 대리 입력 | 구현 단순 | 주관성 개입, 신뢰도 낮음 |
| 자기보고(설문) | 구현 가장 단순 | 정확도 매우 낮음 |

### 6-2. 처방 모듈 (RQ2)

```
[진단 결과]
    │
    ├── 독자 유형 분류 (애독자/간헐적/비독자)
    │           +
    ├── 읽기 수준 (유창성 + 독해 점수 → 레벨화)
    │
    └── 매트릭스 기반 처방 생성
            ├── 적합 텍스트 매칭 (텍스트 풀에서 선택)
            ├── 완독 경험 설계 (목표 텍스트 길이/수준 조정)
            └── 독서 활동 처방 (장르, 읽기 방식 제안)
```

### 6-3. 리포팅 모듈 (RQ3)

**리포트 대상 및 목적**:
| 대상 | 제공 정보 | 목적 |
|------|-----------|------|
| 학생 | 내 읽기 수준, 강점/약점, 다음 목표 | 자기 이해 및 동기 부여 |
| 부모 | 자녀 독서 실태, 가정에서의 지원 방법 | 가정 내 독서 환경 개선 |
| 교사 | 학급 전체 읽기 수준 분포, 슬로러너 식별 | 맞춤 지도 계획 수립 |

**이순영(2021a)이 지적한 부모 대상 정보 공백**:
- 자녀 독서 지도 시 참고할 정보가 아예 없다는 부모가 11.7%, 고학력 부모에서는 23.1%
- 정보 접근 **능력**의 문제가 아니라 적합한 정보 **자체**가 없는 문제

## 8. 구현 시 핵심 기술 과제

### 8-1. 음독 유창성 측정 방법 결정 (최우선 결정 사항)

**옵션 A: STT 기반 완전 자동화**
- 필요 기술: 한국어 STT (Clova, Google Speech-to-Text 등), 오류 음절 자동 분류 로직
- 장점: 정확하고 확장 가능
- 단점: 어린이 음성 인식 정확도 낮음, 비용 높음
**옵션 B: 하이브리드 (타이머 자동 + 오류 수동 입력)**
- 타이머는 앱이 자동 측정, 오류 수는 부모/교사가 카운트 입력
- 중간 수준의 정확도와 구현 난이도
**옵션 C: 완전 설문 방식 (초기 MVP)**
- "자녀가 1분에 몇 어절 읽나요?" 등의 부모 자기보고 방식
- 정확도 낮지만 빠른 프로토타이핑 가능
### 8-2. 텍스트 풀 구성

- 학년별(초1~중1) × 장르별(이야기글/설명글) × 수준별(상/중/하) 분류 필요
- 이은주(2026)의 7원칙을 충족하는 텍스트 검수 필요
- AI 생성 vs 기존 텍스트 저작권 확보 여부 결정 필요
### 8-3. AI 진단 프롬프트 설계

이순영·김주환(2024) 연구에 따르면:
- AI는 감정이나 선입견 없이 텍스트를 객관적으로 평가 가능
- **단, 구조화된 프롬프트 없이는 진단 품질이 제한적**
- → 사실적/추론적/비판적 이해 각 층위별 평가 기준을 프롬프트에 명시해야 함
### 8-4. 독자 유형 분류 로직

독자 유형(애독자/간헐적/비독자)은 읽기 수준과 별개로 측정 필요:
- 독서 빈도: "지난 1주일에 몇 번 책을 읽었습니까?" 등의 설문
- 독서 태도: 리커트 척도 기반 태도 측정 문항
- 위 두 가지 모두 충족 시 → 애독자 분류 기준

## 9. 관련 교육학 용어 사전

| 용어 | 정의 |
|------|------|
| **슬로러너 (Slow Learner)** | 지능은 정상이지만 학업 성취가 또래보다 느린 학습자. 읽기에서는 유창성 또는 독해 처리 속도가 느린 학생 |
| **이독성 (Readability)** | 텍스트가 얼마나 읽기 쉬운지를 수량화한 지표. 어휘 난도 + 문장 길이 등으로 계산 |
| **음독 (Oral Reading)** | 소리 내어 읽기. 유창성 직접 관찰 가능 |
| **묵독 (Silent Reading)** | 소리 없이 읽기. 속도 빠르지만 측정 어려움 |
| **자동성 (Automaticity)** | 해독 과정에 인지 자원을 소모하지 않고 자동으로 처리하는 능력 |
| **PISA** | Programme for International Student Assessment. OECD의 만 15세 대상 국제 학업성취도 평가 |
| **PIAAC** | Programme for the International Assessment of Adult Competencies. 성인 역량 국제 조사 |
| **GRI** | General Reading Inventory. 읽기 이해 능력 측정 검사 도구 |
| **완독 경험 설계** | 학습자가 책을 끝까지 읽는 성공 경험을 갖도록 난이도와 분량을 의도적으로 설계하는 것 |
| **비독자** | 책을 거의 읽지 않는 사람. 단순히 읽지 않는 것(Non-reader)과 읽기 싫은 것(Aliterate)이 혼재 |
| **적합도서** | 개별 독자의 특성(흥미, 수준, 장르 선호 등)에 맞는 책. '추천도서'와 구분 |
